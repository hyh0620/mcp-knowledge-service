# 会员与售后规则 Membership and After Sales

Version: 2026.07
Updated: 2026-07-07

## 概述

Source document: membership_and_after_sales. Version: 2026.07. Updated: 2026-07-07. Section: 概述.

本文档用于回答会员记录、偏好记录、售后沟通、投诉处理和复购提醒等问题。会员信息用于提升服务连续性，不替代用户每次预约时的明确选择。

价格、服务时长和预约状态仍以业务系统结构化数据为准。

## 会员记录

Source document: membership_and_after_sales. Version: 2026.07. Updated: 2026-07-07. Section: 会员记录.

会员可记录常用发型师、风格偏好、发色记录、护理偏好和历史服务备注，用于后续预约推荐和回访提醒。

例如，顾客常做冷棕色、偏好低维护发型、常指定某位发型师，这些信息可以帮助前台更快沟通。但每次预约仍要由顾客确认服务项目、时间和发型师安排。

## 偏好和隐私边界

Source document: membership_and_after_sales. Version: 2026.07. Updated: 2026-07-07. Section: 偏好和隐私边界.

会员偏好只用于理发店服务连续性，例如发长、发质、发色历史、风格偏好、常用发型师和护理偏好。

系统不应记录 API Key、敏感身份信息或与门店服务无关的私人内容。日志中也不应写入完整密钥或过长用户输入。

## 售后沟通

Source document: membership_and_after_sales. Version: 2026.07. Updated: 2026-07-07. Section: 售后沟通.

染发、烫发和剪发效果与发质基础、参考图差异、日常护理和个人审美有关。服务前建议充分沟通目标效果和可接受范围。

如对服务结果有疑问，建议在离店前或服务后 7 天内联系门店。门店会根据服务记录、现场照片、发型师说明和顾客反馈进行复核。

## 投诉处理

Source document: membership_and_after_sales. Version: 2026.07. Updated: 2026-07-07. Section: 投诉处理.

投诉处理应先确认预约记录、服务项目、服务时间、发型师、沟通过程和顾客反馈。可由门店负责人进行复核，必要时安排补充沟通或合理售后处理。

RAG 文档只提供规则说明，不应自动承诺退款、赔偿或医疗性质结论。

## 回访提醒

Source document: membership_and_after_sales. Version: 2026.07. Updated: 2026-07-07. Section: 回访提醒.

染发后可提醒顾客注意护色、减少高温清洗和暴晒。烫发后可提醒顾客 48 小时内避免洗头和压住卷度。头皮护理后可提醒顾客当天减少强定型产品。

这些提醒属于服务建议，不替代专业医疗意见。

## 常见问答

Source document: membership_and_after_sales. Version: 2026.07. Updated: 2026-07-07. Section: 常见问答.

问：会员会记录哪些信息？
答：常用发型师、风格偏好、发色记录、护理偏好和历史服务备注。

问：服务后不满意怎么办？
答：建议在离店前或服务后 7 天内联系门店，门店会根据记录和反馈复核。

问：系统会自动决定售后赔偿吗？
答：不会，售后处理需要门店人工确认。
