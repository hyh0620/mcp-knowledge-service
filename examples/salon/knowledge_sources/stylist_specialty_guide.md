# 发型师专长指南 Stylist Specialty Guide

Version: 2026.07
Updated: 2026-07-07

## 概述

Source document: stylist_specialty_guide. Version: 2026.07. Updated: 2026-07-07. Section: 概述.

本文档用于解释发型师专长、系统推荐依据和可选偏好边界。它不决定具体预约是否成功，最终仍由业务数据库中的排班和冲突校验决定。

门店发型师专长覆盖男士短发、渐变推剪、女士层次、日系造型、染发调色、烫发造型、儿童剪发、活动造型和头皮护理。

## 男士短发和渐变推剪

Source document: stylist_specialty_guide. Version: 2026.07. Updated: 2026-07-07. Section: 男士短发和渐变推剪.

男士短发和渐变推剪注重轮廓、鬓角、后颈线条和日常清爽度，适合通勤、学生和运动风格。

顾客可以描述偏好，例如清爽、商务、寸头、油头、渐变层次明显或自然过渡。系统会优先匹配擅长男士短发和渐变推剪的发型师。

## 女士层次和日系造型

Source document: stylist_specialty_guide. Version: 2026.07. Updated: 2026-07-07. Section: 女士层次和日系造型.

女士层次和日系造型注重轻盈感、脸型修饰、刘海细节和自然吹整效果。

如果顾客希望低维护、自然、修饰脸型或保留发长，应提前说明当前发长、是否接受明显层次和日常打理时间。

## 染发调色专长

Source document: stylist_specialty_guide. Version: 2026.07. Updated: 2026-07-07. Section: 染发调色专长.

染发调色注重目标色沟通、肤色匹配、发质评估和褪色预期管理。

适合冷棕色、显白发色、补染、挑染和通勤发色。发型师会结合底色、历史染烫和是否接受漂发给出建议。

## 烫发造型专长

Source document: stylist_specialty_guide. Version: 2026.07. Updated: 2026-07-07. Section: 烫发造型专长.

烫发造型注重卷度、蓬松度、发量平衡和后续打理难度。

适合纹理烫、发根蓬松、男士卷发、女士卷发和需要改变发尾走向的顾客。发质严重受损时可能建议先护理。

## 儿童剪发和头皮护理

Source document: stylist_specialty_guide. Version: 2026.07. Updated: 2026-07-07. Section: 儿童剪发和头皮护理.

儿童剪发注重沟通耐心、动作稳定和安全体验，适合亲子预约。

头皮护理专长关注控油清洁、敏感头皮护理、洗护护理和造型产品残留清洁。严重头皮问题应以医疗机构建议为准。

## 推荐依据

Source document: stylist_specialty_guide. Version: 2026.07. Updated: 2026-07-07. Section: 推荐依据.

系统推荐发型师时会综合服务类型、风格偏好、预算、可预约时间、发型师专长和用户评分。

指定发型师是可选偏好，不是必填项。用户不指定发型师时，系统会按服务和偏好推荐可预约人选。性别偏好也是可选项，理发店预约不要求用户必须提供性别偏好。

## 常见问答

Source document: stylist_specialty_guide. Version: 2026.07. Updated: 2026-07-07. Section: 常见问答.

问：男士短发应该选什么专长？
答：优先匹配男士短发、渐变推剪、商务短发或寸头专长。

问：染发应该选什么专长？
答：优先匹配染发调色、发色设计、冷棕色或补染专长。

问：不指定发型师可以预约吗？
答：可以，系统会根据服务、偏好和可预约时间推荐。
